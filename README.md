# Todolist

A Todolist Project Developed With React, Redux And Tailwindcss


## Overview

In This Project, The User Can Add Tasks, Change Them To "Done" Or "Undone" Status, Or Delete Them.


## Tech Stack

* React
* Redux
* Tailwind CSS


## Installation

Clone the repository:

```bash
git clone https://github.com/AliEnsafi/todolist-with-redux.git
```

Install dependencies:

```bash
npm install
```

Run development server:

```bash
npm run dev
```

Open:

```text
http://localhost:5173/
```
